# 05 — Staff Training Materials (P7 Phase 5)

Nine role-based guides plus SOPs and quick-reference cards. Each guide covers:
daily routine, key screens, common tasks, error handling, and escalation.

> These are the trainer-facing masters. Print the Quick-Reference section for
> each station. Deliver classroom + hands-on session using the UAT scenarios
> (04) as exercises.

## Role Guides
1. **Owner Guide** — dashboard, financial health, approvals, user management,
   reports, AI Copilot executive queries, backup/restore oversight.
2. **Manager Guide** — shift open/close, voids/refunds authority, night audit,
   inventory adjustments, supplier payments, workflow approvals, parallel-run
   reconciliation review.
3. **Cashier Guide** — login (PIN/biometric), sales entry, split/tender,
   cash drawer, refunds (with manager approval), end-of-shift report.
4. **Restaurant Guide** — open table/session, take order, send KOT to kitchen,
   modify/void lines, bill, apply discounts, settle.
5. **Bar Guide** — bottle open, peg definitions, bar sale, excise register
   entries, stock out, reconciliation.
6. **Reception Guide** — availability, booking, check-in/check-out, folio,
   guest billing, room status, deposits/credit.
7. **Kitchen Guide** — KDS view, accept/ready/bump KOT, station routing,
   delay/recall, wastage logging.
8. **Accountant Guide** — chart of accounts, journals, auto-posting, GST
   returns data, bank reconciliation, trial balance, closing.
9. **Administrator Guide** — user/role management, device config, backup
   schedule, monitoring alerts, tenant/feature flags, update procedure.

## Standard Operating Procedures (SOPs)
- **SOP-01 Open Shift:** unlock drawer, verify float, sign in, print Z-base.
- **SOP-02 Close Shift:** reconcile cash, run end-of-shift, manager approval,
  lock drawer.
- **SOP-03 Void/Refund:** manager PIN, reason code, audit log entry.
- **SOP-04 Stock Receipt:** GRN against PO, scan barcode, discrepancy report.
- **SOP-05 Night Audit:** run audit, verify room+restaurant+bar totals tie to
  bank, sign off.
- **SOP-06 Backup:** confirm scheduled backup succeeded, verify off-site copy.
- **SOP-07 Incident:** log in defect log, notify admin, workaround, escalate.
